PlantUML diagram for LittleZelda

Render instructions:

- Install PlantUML (example via Homebrew):

```bash
brew install plantuml
```

- Render PNG from the `.puml` file:

```bash
plantuml diagrams/classes.puml
```

- Or render to SVG:

```bash
plantuml -tsvg diagrams/classes.puml
```
